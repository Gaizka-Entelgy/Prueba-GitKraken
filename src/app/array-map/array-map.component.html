-map()-

El método map() crea un nuevo array con los resultados de la llamada a la función indicada aplicados a cada uno de sus elementos. Es decir, pasa uno por uno por cada una de las posiciones de un array original y ejecita las operaciones indicadas.

El array original es: [1, 5, 10, 15]

Introduce un número y el método map() multiplicará todos los valores por dicho número.
